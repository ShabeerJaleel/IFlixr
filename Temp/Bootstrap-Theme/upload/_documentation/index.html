Color schemes usage

You got 18 unique color schemes. To setup is fairly easy. You need to add sheme styles to your HTML. There two ways. You can set it globally (for all pages) or locally (for one page). 

To set scheme globally, you need to open mystyles.css file. It's a blank file for all your custom styles. I highly recommend you not to change core styles and use mystyles.css for customization.

Inside mystyles.css type

@import "/schemes/contessa.css"

contessa.css is the name of color scheme. You can find all schemes in css/schemes directory

 

To set scheme locally (only for one page) Insert sheme styles directy in your header. 

Inside <head> of your file type

<link rel="stylesheet" href="css/shemes/contessa.css">

And again contessa.css is the name of scheme. Choose whatever you'd like

 

 

Body customization

You can set 1200px or 960px grid and also boxed or wide layout. Here is little tutorial.

By default <body> has width1200 class. So to set 960px grid, simply rid off width1200px class from <body>

To setup boxed layout, simply add boxed class to <body> tag. <body class="boxed">.

You can mix grid width and boxed-wide layout

<body class="width1200">   Default layout with 1200px grid width and wide 

<body>  960px grid wide

<body class="width1200 boxed> 1200px grid boxed

<body class="boxed"> 960px grid boxed

Background Images and Patterns

You can use this technique both for <body> or any element on your page.

To set full background image, add bg-img class to your element, and after add prefered image via inline style

<div class="bg-img" style="background-image: url('img/bg/my-background-image.jpg')>

There is sweet shortcut 

<div data-bg-img="img/bg/my-background-image.jpg">

Both of them do the same thing. If you will use second shortcut method, you don't have to add bg-img class to your element.

 

It's pretty similar to setup background pattern

<div class="bg-pattern" style="background-image: url("img/patterns/my-pattern.jpg")>

or

<div data-bg-pattern="img/patterns/my-pattern.jpg">

Smart Mask Usage

You can place mask between objects. And control actual active and unactive(under mask) layers.

To set the mask create a <div> with bg-mask class

<div class="bg-mask">

Everything inside this <div> will be inside the mask. You can set the mask color and aplha level.

for example <div class="bg-mask-lighten bg-mask-color">

We just create color mask (by color I mean color sheme main color) with light (high transperenty) mask.

Everything inside the mask becore unclickable.

To set front content (active content) create a <div> with class mask-front.

This technique are perfect for background images dim. 

for example

<div class="bg-mask>

   <div data-bg-img="my-bg-image.jpg></div>

   <div class="mask-front"> My active content </div>

</div>

 

You have 3 mask colors black(default), white, and color (rely on color scheme)

Live device Mockups

We All like Device mockups, but you require to change photoshop files to change an image. It will be cool, if you can do it with code. Now it's possibe!

Create a <div> with class device-mockup and set desirable device.

<div class="device-mockup" data-device="iphone5">

</div>

some devices support alternative color and orientation

<div class="device-mockup" data-device="iphone5" data-color="white" data-orientation="landscape">

</div>

You can Insert ANY content inside the mockup. By any I literally mean ANY video, images. sliders, custom html, google maps... everyghing your want.

<div class="device-mockup data-device="iphone5>

   <div class="device>

        <div class="screen">

           AMY CONTENT HERE!

         </div>

     </div>

</div>

 

You got  19 mockups. You can find them in img/device-mockups directory

Icon effects

We have TONS of icons effects which you can use with any of 361 fontawesome icon.

The basic setup is just add box-icon to your icon

<i class="icon-heart box-icon></i>

You can set size of icon (big, large, huge, normal)

<i class="icon-heart box-icon-large></i>

Also you can set a color (normal, inverse, black, gray, info, success, warning, danger, white)

<i class="icon-heart box-icon-success"></i>

Setup a hover color (color will be changed on hover)

<i class="icon-heart box-icon-to-black"></i>

Shape (round, border, border-dashed)

<i class="icon-heart box-icon-border"></i>

Animation (Flash, Shake, Bounce, Tada, Swing, Pulse, Border rise, Border rise alt, Border fade out, Border fade in, Left to right, Right to left, Top to bottom, Bottom to top)

<i class="icon-heart animate-icon-left-to-right'></i>

Aligment (left, right, center)

<i class="icon-heart box-icon-left></i>

Mix any of the effects to get a desirable icon

<i class="icon-heart box-icon-huge box-icon-black box-icon-to-success box-icon-round animate-icon-flash"></i>

 

So we have unlimited icon effects :)
